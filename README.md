Run command for install dependencies

### `npm i`

or

### `yarn install`

In this project used the json-server
So you need to install

### `npm i json-server -g`

or

### `yarn global add json-server`

Then in the project directory run:

### `json-server db.json`

After that, you can start webpack-dev-server

### `npm start`

or

### `yarn start`
